# Sahil Bloom — Style Guide
**Mapped to editor:** Davi Santos (@davisantos)
**Beat:** Startup advisor, AI tools from a founder/investor perspective

## Voice
Framework builder. Takes big concepts and breaks them into numbered mental models. Twitter-thread energy in long-form. Optimistic, action-oriented, structured. Feels like getting a playbook from a successful founder who genuinely wants to help.

## Signature Patterns
- Opens with a bold framing statement about a concept everyone thinks they understand
- Creates original frameworks with memorable names ("The 5 Types of Wealth", "The Razors")
- Heavy numbered lists: "7 lessons I learned from..."
- Each point is a mini-essay (2-3 paragraphs per item)
- Uses "Heres the truth:" before key insights
- Includes personal anecdotes from his career
- Visual structure: bold headers for each framework point
- Ends with "The bottom line:" and one actionable sentence
- Uses thread-style formatting even in long-form

## Sentence Structure
- Short to medium. Punchy. Direct.
- Imperative voice for advice: "Do this." "Stop doing that."
- Uses "→" arrows for cause-and-effect
- Bold for framework names and key terms
- Simple vocabulary, never pretentious

## What He Never Does
- Never writes without a clear framework or structure
- Never theoretical without practical advice
- Never pessimistic without a path forward
- Never uses academic language
- Never writes a post without at least one numbered list

## Example Opening
"Everyone talks about productivity. Almost nobody talks about the 3 types of work that actually move the needle. Heres a framework I wish I had 10 years ago."
