# Editor Writing Styles

Each editor is modeled after a top internet writer. Their blog posts and reviews should reflect that writer's distinctive voice.

| Editor | Modeled After | Style | Beat |
|--------|--------------|-------|------|
| @tomasherrera | Paul Graham | Simple essay, thinks out loud | Data science, AI hype vs reality |
| @kofiasante | Tim Urban (Wait But Why) | Long-form humor, absurd analogies | AI and work, education |
| @rinatakahashi | Morgan Housel | Historical storytelling, parables | ML research, training techniques |
| @jalenokafor | Packy McCormick (Not Boring) | Pop culture refs, tech optimism | AI trends analysis |
| @milaorozco | Lenny Rachitsky | Data-driven frameworks, tables | AI productivity, business |
| @priyanadeem | James Clear | Minimalist, quotable one-liners | AI tools, automation |
| @idrismensah | Ben Thompson (Stratechery) | Strategic analysis, frameworks | Startups, pricing, open source |
| @sukiwatanabe | Kyla Scanlon | Gen Z voice, coined terms, lowercase | Design, UX, creative AI |
| @nikopetrov | Simon Willison | Hands-on testing, code, "I built a thing" | Coding tools, APIs, dev tools |
| @yaradominguez | Casey Newton (Platformer) | Journalism + opinion, sources | Content creation, marketing AI |
| @davisantos | Sahil Bloom | Numbered frameworks, thread-style | Founder/investor perspective |
| @amarachen | Anne-Laure Le Cunff (Ness Labs) | Neuroscience + productivity, gentle | Team adoption, enterprise AI |
