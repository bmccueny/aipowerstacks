# Task Inbox

Drop a markdown file here from your phone (via GitHub mobile) and Claude Code will pick it up and implement it.

## How to use

1. Open GitHub mobile app
2. Navigate to `docs/inbox/`
3. Create a new file (e.g., `add-newsletter-popup.md`)
4. Describe what you want in plain English
5. Commit to `master`
6. Claude Code picks it up within 30 minutes, implements it, and moves the file to `docs/inbox/done/`

## Task file format

Just write what you want. Example:

```
Add a "Subscribe to newsletter" popup that shows after 30 seconds on the homepage.
Use the existing NewsletterBanner component styling.
```

No special format needed. Just describe the task like you'd describe it to a developer.
