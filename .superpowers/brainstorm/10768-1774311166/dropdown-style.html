What changes across the site

All 4 dropdown types will get the same treatment:

  • DropdownMenu (user avatar, share, challenge submit) → liquid glass panel
  • Select (tool filters: category, pricing, sort) → liquid glass panel
  • AddToStack (custom portal dropdown) → liquid glass panel + bottom sheet on mobile
  • Mobile menus (navbar, admin) → liquid glass bottom sheet / drawer
