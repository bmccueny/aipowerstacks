## Everything you need, nothing you don't

No config files. No build steps. Just results.

###### /doc-smith-create

Analyze your codebase and generate a complete, navigable documentation site with AI-powered diagrams.

```
$ /doc-smith-create

Analyzing project structure...
Generating documentation...
[DONE] 12 pages created in ./docs
```

###### /doc-smith-localize

Translate your documentation into any language with consistent terminology and structure.

```
$ /doc-smith-localize --lang zh,ja,ko

Translating to 3 languages...
[DONE] Localization complete
  → zh: 12 pages
  → ja: 12 pages
  → ko: 12 pages
```

###### /doc-smith-publish

Deploy your documentation to the cloud with a single command. Instant public URL.

```
$ /doc-smith-publish

Packaging documentation...
Deploying to cloud...
[DONE] Published!
  → https://my-project.docsmith.aigne.io
```

## Built for engineers, loved by teams

Ship documentation as fast as you ship code.

###### Repo-Aware Analysis

Understands your project like a senior engineer — catches APIs, configs, and edge cases.

###### Structured Planning

Generates a doc outline for your review before writing a single page.

###### Clean HTML Output

Produces navigable, searchable documentation sites ready for your team.

###### Automatic Architecture Diagrams

Architecture charts, flowcharts, and data models — generated automatically.

###### Multi-Language

Translate to any language while keeping terminology and structure consistent.

###### Incremental Updates

Only regenerates what changed — keeps docs in sync without full rebuilds.
