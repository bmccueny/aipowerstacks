# Pricing

Lightfield supports teams of all sizes, from company inception to growth stage.

## Startup

For companies doing founder sales.

[Try for Free](https://crm.lightfield.app/signup)

$59 per user per month

Billed monthly

Features:

- Call intelligence - record, transcribe, and analyze calls
- Automated data enrichment and record updates
- Unlimited agent queries and actions
- Fully configurable data model

## Pro

For scaling companies.

[Request a demo](https://lightfield.app/book-demo)

$149 per user per month

Billed annually

Features:

- Increased limits for records and workflow events
- Advanced group configuration and permissioning
- White glove migration, onboarding and support
- Dedicated customer success manager
