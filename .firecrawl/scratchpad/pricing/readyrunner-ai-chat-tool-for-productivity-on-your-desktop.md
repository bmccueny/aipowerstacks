## Questions

### What is the Dev plan?

The Dev plan is for people who have their own OpenAI developer API key. Messages will be sent using your key, and ReadyRunner won't impose message limits. You will be charged by OpenAI for your API usage. You will only have access to models that your OpenAI API (not ChatGPT) account has access to.

### What is GPT-3.5?

GPT-3.5 is OpenAI's popular AI chat model which can generate realistic responses, helping you write, edit, generate code, etc.

### What is GPT-4?

GPT-4 is the latest model from OpenAI. It understands more complex language and often generates more accurate responses and better code.

### What counts as a message?

One message sent to the AI which receives a response. If there is an error generating the response, it won't count.

### What if I reach the message limit?

When you reach your message limit, you can either wait until the next month or upgrade to a higher plan. Or you can buy additional message credits.
