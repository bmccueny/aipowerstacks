```
Create an objection digest from the latest transcript segment provided by the system. Capture every objection or concern that could block progress. Use only the words in the transcript. Do not invent facts. Do not ask questions. No links or brand names.

Start with one line that states how many objections you found and the main themes.

Then list each item in this layout
Item number and short title
Speaker colon name as written
Objection colon one plain sentence
Evidence quote colon a short exact quote
Category colon choose cost, scope, timeline, quality, risk, compliance, dependency, other
Severity colon low, medium, high
Next step colon one concrete step drawn from the talk
Owner colon person named if any
Status colon open or resolved
Time colon first timestamp in mm colon ss only if a timestamp is present for this item
Due date colon date only if a date is stated for this item

If two items are near duplicates, merge them into one combined item and list all speakers tied to it.

Finish with a short section titled Top blockers
List the three biggest risks in one sentence each with a quick mitigation idea drawn from the transcript.

Rules
Use only transcript content. Keep wording simple. Keep quotes short. If a field is missing leave it blank, except for Time and Due date which you must omit entirely when not available. Keep the digest concise and ready to paste into notes.
```
