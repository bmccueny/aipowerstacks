## Overview

DeepTutor is an AI-powered Personalized Learning Assistant designed to transform the interaction of students with educational materials. Built upon a unified multi-agent architecture, the tool offers features designed to enhance the educational experience.

Users can upload textbooks, papers, and manuals to build AI-powered knowledge repositories with Retrieval-Augmented Generation (RAG) and knowledge graph integration.

The tool is capable of transforming complex concepts into visual aids with step-by-step breakdowns for better understanding. It also features a smart problem solver that utilizes RAG, web search, paper search, and code execution to provide step-by-step solutions.A distinctive feature includes the generation of targeted quizzes and practice problems while also allowing users to upload reference exams to match the original style and difficulty.

In addition, DeepTutor helps in comprehensive topic exploration with systematic analysis and literature review. Another striking feature is the facilitation of automated and interactive concept development grounded in multi-source insights and novelty evaluation.For improved comprehension, DeepTutor makes use of semantic connections powered by LightRAG.

The platform supports multiple modalities such as PDF, LaTeX, images, and code execution. DeepTutor is not just a problem-solving platform, it is also a question generation, research, and idea generation platform, thereby making it a comprehensive tool for learning and teaching.

The tool is open-source and the scripts for installation and starting the application are readily available.
