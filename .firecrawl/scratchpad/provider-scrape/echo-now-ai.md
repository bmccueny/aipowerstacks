## Frequently Asked Questions

Everything you need to know about Echo Now and how it can transform your
Slack experience.

What is Echo Now?

Echo Now is an AI-powered Slack assistant that helps teams stay productive by analyzing conversations, creating intelligent summaries, and providing interactive chat capabilities directly within Slack. Never miss important updates again.

How does the daily summary feature work?

Echo Now automatically generates personalized daily summaries delivered every morning at 9 AM in your local timezone. These include important channel activities, action items requiring your attention, key decisions, and upcoming deadlines - all intelligently filtered to focus on your working days.

What types of action items does Echo Now identify?

Echo Now identifies direct questions addressed to you, requests for information or decisions, task assignments, deadline notifications, and unanswered questions where you're expected to respond. It smartly avoids creating action items for general announcements or messages you've already addressed.

Can I get summaries for specific channels?

Yes! You can request on-demand channel summaries by asking the AI assistant for recent activities in any channel. Echo Now analyzes recent messages and provides comprehensive overviews including key topics, decisions, and action items specific to that channel.

How does the interactive chat feature work?

The AI chat assistant provides context-aware responses about your Slack conversations. Ask questions about discussions, get help understanding complex topics, and receive real-time streaming responses - all without leaving Slack.

Is there a usage limit?

Echo Now has a daily limit designed to accommodate typical team usage patterns. This ensures fair usage and system stability while meeting the needs of most teams.

How does Echo Now ensure data privacy and security?

Your privacy is our priority. Echo Now is built with enterprise-grade security, follows Slack's security best practices, uses OAuth 2.0 standards, and processes data securely without permanent storage of your message content.

What platforms and integrations are supported?

Echo Now fully integrates with Slack (channels, DMs, threads) and supports Model Context Protocol (MCP) servers for extended functionality with external tools and task management systems.

How do I install Echo Now for my team?

Installation is simple: add the Echo Now app to your Slack workspace, complete the OAuth authorization, and the app automatically starts analyzing messages and providing summaries. Your team can immediately use all features.

Can Echo Now work with private channels?

Yes, Echo Now can access the same channels that you have access to, including private channels. It can only analyze channels where you already have viewing permissions.
