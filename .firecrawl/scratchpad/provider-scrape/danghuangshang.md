常见问题

## 有问有答

要花多少钱？+

软件免费开源（MIT 协议）。你需要一个 Anthropic API Key（约 $5-20/月）和一台服务器。Oracle Cloud 有永久免费的 ARM 实例可以用。

可以自定义角色吗？+

可以。只需要在配置文件里添加新 Agent 即可，不用改代码。翰林院、太医院、御膳房……想加什么加什么。

支持哪些 AI 模型？+

默认优化 Claude（Anthropic）。架构支持切换到 GPT-4、通义千问等其他模型。

数据安全吗？+

所有数据跑在你自己的服务器上，除了 AI 模型 API 外不会发送到任何第三方。

可以商用吗？+

可以。MIT 开源协议，商用无限制。

和 ChatGPT 有什么区别？+

ChatGPT 是单个助手。当皇上给你一整个内阁 —— 多个专业 Agent 互相协作、记忆上下文、定时任务、集成 Discord。

不想用 Discord 行不行？+

行。OpenClaw 自带 WebUI，装好直接浏览器打开就能聊。Discord、飞书、WhatsApp 都是可选通道。
